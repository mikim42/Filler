# Filler

__Installation:__

* `git clone https://github.com/mikim42/Filler.git`
* `make`

**Usage:**
* `./filler_vm -f <MAP_FILE> -p1 <PLAYER> -p2 <PLAYER>`
* `./filler_vm --help`

**Example:**
* `./filler_vm -f source/maps/map00 -p1 ./filler -p2 source/players/superjeannot.filler`

**Keys:**
* `ESC`: exit
